## 🚀 Como Usar

### Localmente
```bash
cd boxtiweb
python3 -m http.server 8080
# Acesse: http://localhost:8080
```

### Deploy
1. Copie os arquivos para seu servidor web
2. Configure o domínio
3. Pronto! Site está online
